# App Quickstart

Use the HonestRoles app to run workflows from your browser.

## When to use

Use this when you want the fastest path to value without local setup.

## Prerequisites

- A web browser
- Access to [https://honestroles.com](https://honestroles.com)

## Steps

1. Open [https://honestroles.com](https://honestroles.com).
2. Create or sign in to your account.
3. Start a new workspace or open an existing one.
4. Follow the app prompts to upload data and run your first analysis.

## Expected result

You can create a workflow in the app and access analysis outputs from the browser.
