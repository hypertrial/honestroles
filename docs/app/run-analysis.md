# Run Analysis

Execute a complete HonestRoles workflow in the app.

## When to use

Use this after your data is uploaded and validated.

## Prerequisites

- Data uploaded in the app
- Workspace access at [https://honestroles.com](https://honestroles.com)

## Steps

1. Open your dataset in the app.
2. Select workflow options and run analysis.
3. Wait for run completion and review summary metrics.
4. Open ranked results and diagnostics views.

## Expected result

You get ranked output and diagnostics for the run in the app interface.
