# Upload Data

Prepare and upload job data to the HonestRoles app.

## When to use

Use this before running an analysis in the app.

## Prerequisites

- Access to [https://honestroles.com](https://honestroles.com)
- A dataset ready for upload

## Steps

1. Open your workspace in the app.
2. Navigate to the data upload area.
3. Upload your dataset and confirm the import finishes.
4. Review detected columns and data health summary.

## Expected result

Your dataset is available in the app and ready for analysis runs.
