# Export Results

Export analysis output from the HonestRoles app for downstream use.

## When to use

Use this after a successful analysis run.

## Prerequisites

- Completed run in [https://honestroles.com](https://honestroles.com)

## Steps

1. Open the run results page.
2. Choose export format and destination.
3. Download or copy exported output.
4. Validate exported output in your downstream workflow.

## Expected result

You have portable output artifacts from the app run.
